Portfólio pessoal com identidade visual cyberpunk, desenvolvido para apresentar meus projetos, experiências e links profissionais de forma moderna e impactante.

Profissional de T.I desde 2017, atualmente em transição estratégica de Infraestrutura e Cloud para Desenvolvimento Web, unindo base sólida em infraestrutura com construção de aplicações modernas.

---

## Sobre o Projeto

Este projeto foi criado para consolidar minha evolução profissional de Infra/Cloud para Desenvolvimento Web, integrando:

- Mentalidade orientada a infraestrutura e performance  
- Experiência prática com AWS e GCP  
- Automação de processos com Python  
- Manipulação e análise de dados com SQL  
- Desenvolvimento Front-End moderno  

A proposta é oferecer uma experiência visual imersiva, com estética tech/cyber e foco em organização, clareza e performance.
